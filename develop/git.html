    Git 实用技巧¶

    📖 阅读信息

    阅读时间：1 分钟 | 中文字符：266 | 有效代码行数：48

    ⚡ AI摘要 (GLM)

    本文介绍了Git的基本操作，如新建仓库、克隆、提交、分支管理及合并冲突处理。还涵盖了一些高级技巧，如临时保存、灵活合并分支、 cherry-pick、修改提交、取消修改和弃用提交等。

    1. 新建 git 仓库¶

    1
    git init

    8. 删除分支¶

    1
    git branch -d test-not-need

    9. 合并冲突¶

    当两个分支都对同一行进行了修改，git 便会产生冲突，并标记为未合并

    此时将每个文件进行修改，确认最后的内容，使用 git add 方法标记为冲突已解决 

    1
    git add .\A.txt
    在所有文件的冲突均已解决后，使用 commit 提交此次修改。

    1
    git merge --abort
