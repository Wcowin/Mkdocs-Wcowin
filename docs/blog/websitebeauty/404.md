首先在mkdocs.yml文件中添加custom_dir:
``` hl_lines="3"
theme:
  name: material
  custom_dir: docs/overrides #覆写路径
```

docs/overrides文件下新建404.html  
树状结构如下:  
``` hl_lines="11"
$ tree -a
.
├── .github
│   ├── .DS_Store
│   └── workflows
│       └── PublishMySite.yml
├── docs
│   └── index.md
│   └──overrides
│       └──assets
│       └──404.html
│       └──main.html
│       └──partials
│          └──comments.html
│
└── mkdocs.yml
```
