    添加404页面

    📖 阅读信息

    阅读时间：1 分钟 | 中文字符：131 | 有效代码行数：29

    首先在mkdocs.yml文件中添加custom_dir: 

    theme:
  name: material
  custom_dir: docs/overrides #覆写路径

    docs/overrides文件下新建404.html
    树状结构如下:

    $ tree -a
.
├── .github
│   ├── .DS_Store
│   └── workflows
│       └── ci.yml
├── docs
│   └── index.md
│   └──overrides
│       └──assets
│       └──404.html
│       └──partials
│          └──comments.html
│
└── mkdocs.yml
